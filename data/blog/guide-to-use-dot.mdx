## What is dot?

Dot is a very powerful compiler which can convert plain text into images.

You can use it to generate various graphs.

## How to use dot?

First, download it from [Official website](https://graphviz.org/download/).

Then, try write a `test.dot`.

```Dot
digraph test {
    randir=TB; // From top to bottom. You can alse select BT、LR(from left to right)、RL.
    node [shape=box, fontname="Source Han Sans CN"]; // shape=box、diamond...
    edge [fontname="Source Han Sans CN"];
    // The above is the default configuration for nodes and edges.

    A [label="开始测试", style=rounded];
    B [label="输入测试数据", shape=diamond];
    C [label="正在测试"];
    D [label="测试结束", style=rounded];

    A -> B -> C -> D;
}
```

```bash
dot -Twebp test.dot -o test.webp
#dot -Tpng test.dot -o test.png
```

And you may want to write some complex graphs.

```Dot
digraph test123 {
    a -> b -> c;
    a -> {x, y};
    b [shape=box];
    c [label="hello\nworld", color="blue", fontsize=14];
    a -> z [label="hi", weight=100];
    x -> z [label="multi-line\nlabel"];
    edge [style=dashed,color=red];
    b -> x;
    {rank=same; b x}
}
```
